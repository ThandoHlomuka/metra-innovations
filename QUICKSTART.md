# Quick Start - Deploy in 5 Minutes! ⚡

## Option 1: Deploy to Vercel (Fastest - Recommended)

### 1. Push to GitHub (2 minutes)

Open Command Prompt in the project folder and run:

```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/metra-innovations.git
git push -u origin main
```

**Replace `YOUR_USERNAME` with your GitHub username!**

### 2. Deploy to Vercel (1 minute)

1. Go to **[vercel.com](https://vercel.com)**
2. Click **"Sign Up"** (use GitHub)
3. Click **"Add New Project"**
4. Select your `metra-innovations` repository
5. Click **"Deploy"**

**Done!** Your site will be live at `https://metra-innovations.vercel.app`

---

## Option 2: Manual Deployment

### Using Local Server (for testing)

```bash
# Python
python -m http.server 8000

# Node.js
npx http-server -p 8000

# Then open: http://localhost:8000
```

### Deploy to Netlify

1. Go to [netlify.com](https://netlify.com)
2. Drag and drop the project folder
3. Done!
